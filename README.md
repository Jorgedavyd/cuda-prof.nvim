# cuda-prof.nvim
A NVIDIA CUDA Profiling Tools wrapper for Neovim.

# Requirements
`nvcc`: NVIDIA Compiler. \
`nvvp`: NVIDIA Visual Profiler \
`ncu`: NVIDIA Nsight Compute \
`nsys`: NVIDIA Nsight Systems

# Installation
## Lazy
```lua
return {
    "Jorgedavyd/cuda-prof.nvim",
    config = function() end,
    keymap = {}
}
```
